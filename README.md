# Portal de Inducción IMLS 2026

Aplicación web progresiva (PWA) para la inducción de nuevos funcionarios de la Ilustre Municipalidad de La Serena.

## Características
- Diseño "Smart City" moderno y responsive.
- Flujo de 13 pasos interactivos.
- Integración con videos y contenido multimedia.
- Cuestionario de validación de conocimientos.
- Generación de certificado digital.
- Asistente Virtual con IA (Gemini).

## Tecnologías
- React + Vite
- Tailwind CSS
- Lucide React (Iconos)
- Google Gemini API

## Cómo subir a GitHub

Para subir este proyecto a tu repositorio `vecinoslaserenachile-cloud/portal-induccion-imls`, sigue estos pasos en tu terminal local:

1. **Inicializar Git (si no lo has hecho):**
   ```bash
   git init
   ```

2. **Agregar los archivos:**
   ```bash
   git add .
   ```

3. **Hacer el primer commit:**
   ```bash
   git commit -m "Versión inicial del Portal de Inducción IMLS 2026"
   ```

4. **Renombrar la rama a main:**
   ```bash
   git branch -M main
   ```

5. **Agregar tu repositorio remoto:**
   ```bash
   git remote add origin https://github.com/vecinoslaserenachile-cloud/portal-induccion-imls.git
   ```

6. **Subir los cambios:**
   ```bash
   git push -u origin main
   ```

> **Nota:** Asegúrate de tener permisos de escritura en el repositorio y de haber configurado tus credenciales de GitHub en tu máquina local.
